Code review with code quality in mind

Critical System Visualization

ViDI provides 3D visualization of the reviewed system overlayed with critic repors provided by statical code analysis tools.

Issue Resolution Support

Source code can be edited for within ViDI. Some critics can be automatically fixed. Some critics are assocciated with dedicated UIs that improve the resolution.

Overview of a Review

Each review session is recorded and can be overviewed later. Two views show nunber of critics in time and impact of the changes made to code.
